# yt-vsr

yt video search ```(yt-vsr)``` allows you search YouTube videos 



# Installation

``` pip
npm install yt-vsr 
```







# Usage

``` js
const ytsearch= require("yt-vsr")
let results = await ytsearch("Black Panther")
 console.log(results[0].title)
 
    
     Marvel Studios' Black Panther - Official Trailer
 ```
